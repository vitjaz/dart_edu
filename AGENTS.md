# Агент-контекст: dart_edu

## Назначение проекта

Этот репозиторий отражает прогресс в изучении языка программирования Dart.
Каждый коммит — это шаг в обучении: новая тема, разобранный концепт, практическое задание или эксперимент.

## Структура репозитория

- `cli/` — Dart CLI-проект, созданный через `dart create -t cli`.
  - `bin/cli.dart` — точка входа.
  - `lib/cli.dart` — библиотечный код.
  - `test/cli_test.dart` — тесты.
- `readme.md` — краткое описание репозитория.
- `AGENTS.md` — этот файл (контекст для AI-агентов).

## Рабочие практики

- Изменения коммитятся в GitHub по мере прохождения материала.
- Код пишется на Dart, анализируется через `dart analyze`.
- Тесты запускаются через `dart test`.
